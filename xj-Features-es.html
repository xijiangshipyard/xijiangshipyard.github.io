Gameplay: Shapes

The game offers a wide variety of shapes of blocks for players to choose from. Ideal objects can be constructed using blocks of different shapes.

Gameplay: Sizes

Create a variety of objects by setting the size of the blocks.

Gameplay: Materials

Use different materials to express different objects.

Gameplay: Combination (Blueprint)

Combination (Blueprint): an object that can be reused.
This video demonstrates how to create a composite (also known as a blueprint).
Note: The combination needs to be edited once to determine the center position of the combination and facilitate precise placement of the combination into the scene.

Gameplay: Skills

The game provides various skills to facilitate players' creation, and the reasonable use of various skills will bring great convenience.
Attention: Skills need to pass the assessment on a specialized skill learning map before they can be used!!!

Gameplay: Plants

The game provides a rich variety of plant species for players to place. Plants need to be edited on the plant editing page before they can be used.

Gameplay: Character

NPCs are provided in the game for players to set up. Players can create various parts of their characters, such as the head, shoulders, arms, torso, legs, and weapons. Create various roles.

Gameplay: Export and Import

The combinations and character components created in the game can be exported in obj format with UV texture data.
1. It can be exported in the obj format required for 3D printing. In this mode, the exported objects are mixed with different materials, and the exported data does not include UV data.
2. Export to regular obj format, where the exported obj is divided into different sub objects based on the material.

Gameplay: Import/Export Archives

The created archive can be exported as a separate archive file, and other players can import the archive file separately for archive sharing.
